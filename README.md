Web-based tool for StarCraft II bot developers to visualize map navigation grids and plan base layouts. 

Go to [web page](https://samogon-one.github.io/grid-sc2/)

### What you can do:
*   **Wall-Ins**: Design walls to secure your main ramp and natural expansion.
*   **Production Grids**: Design compact layouts that keep unit pathing clear.
*   **Defenses**: Positions for towers and Siege Tanks.
*   **Unit Positions**: Plan defensive setups and staging areas for attacks.
*   **Proxies**: Find the perfect proxy spots.
