Web-based tool for StarCraft II bot developers to visualize map navigation grids and plan base layouts. 

Go to [web page](https://samogon-one.github.io/grid-sc2/)

### What you can do:
*   **Engineer Wall-Ins**: Design walls for the main ramp and the natural expansion.
*   **Production Grids**: Design compact layouts to save base space and maintain clear unit pathing.
*   **Static Defense**: Towers, Siege Tanks
*   **Unit Positioning**: Plan defensive formations and stage points for attack.
*   **Proxy Locations**
